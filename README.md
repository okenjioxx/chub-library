# 📚 Chub Library — SillyTavern Extension

A sleek extension for [SillyTavern](https://github.com/SillyTavern/SillyTavern) that lets you search, browse, and import character cards from [Chub.ai](https://chub.ai) directly within the SillyTavern interface.

---

## ✨ Features

- **🔍 Full Search** — Browse Chub.ai's entire character library with trending, downloads, rating, newest, and random sorting
- **🏷️ Smart Tag Chips** — Autocomplete tag input with chip locking, just like Chub.ai
- **🔞 NSFW / NSFL Toggles** — Control mature content visibility
- **⬇️ One-Click Import** — Import any character with full metadata, tags, and high-res avatar
- **🎨 Premium UI** — Glassmorphism drawer, 3-column grid, floating action buttons, auto-scrolling tags
- **🚀 Fully Standalone** — No backend proxy or server modifications required

---

## 📦 Installation

### Option 1: SillyTavern Extension Installer (Recommended)

1. Open SillyTavern
2. Go to **Extensions** → **Install Extension**
3. Paste this URL:
   ```
   https://github.com/okenjioxx/chub-library
   ```
4. Click **Install** and reload

### Option 2: Manual

```bash
cd <SillyTavern>/data/<user>/extensions/
git clone https://github.com/okenjioxx/chub-library.git
```
Then restart SillyTavern.

---

## 🖼️ Usage

1. Click the **wand icon** (Extensions) at the bottom-left of SillyTavern
2. Click **Chub Library** to open the drawer
3. Browse trending characters or search by name
4. Add tags with the autocomplete chip system
5. Toggle NSFW / NSFL as needed
6. Hover any card → click ⬇️ to import or ↗️ to view on Chub.ai
